# Mortgage Calculator

A calculator that calculates the amortization of a mortgage

## Run Locally

Prerequisites

- Node 14

Clone the project

```bash
  git clone git@github.com:serv/svelte-mortgage.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

Starting Storybook

```bash
  npm run storybook
```
